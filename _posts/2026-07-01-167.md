23년부터 써오던 [Tistory 블로그](https://cornpip.tistory.com) 포스팅을 이전하고, 앞으로 이곳에 글을 작성하려 한다.


라텍스 문법 예시

$$
r_s
=
\frac{
n_1 \cos \theta_i
-
n_2 \cos \theta_t
}{
n_1 \cos \theta_i
+
n_2 \cos \theta_t
}
$$

$r_s$ : s-편광 반사 계수 (Fresnel reflection coefficient)  
$n_1$ : 입사 매질의 굴절률  
$n_2$ : 투과 매질의 굴절률  
$\theta_i$ : 입사각  
$\theta_t$ : 굴절각

## kramdown 확장 문법 + Chirpy 클래스

표준 마크다운에는 없는 문법들이다. 문법 자체는 전부 **kramdown**(주로 IAL `{: ... }`)이고, 붙이는 클래스는 출처가 갈린다.

- **Chirpy 제공 클래스** — `.prompt-tip` / `.prompt-info` / `.prompt-warning` / `.prompt-danger` (콜아웃), `.left` / `.right` / `.normal` (이미지 정렬), `.filepath` (파일 경로)
- **Bootstrap 유틸** (테마에 번들된 것, Chirpy 전용 아님) — `.shadow` (그림자), `.text-center` (가운데 정렬)
- **Chirpy 렌더링 동작** (클래스 아님) — 코드블록 `{: file="..." }` 파일명 바, 이미지 자동 가운데정렬·캡션

즉 `{: .prompt-tip }` 은 "kramdown IAL로 **Chirpy가 정의한 클래스**를 붙이는 것"이다. 아래 소제목의 `(Chirpy)` 표기는 그 스타일/동작이 Chirpy 소속이라는 뜻.

### IAL — 앞 요소에 속성/클래스 붙이기

IAL(Inline Attribute List)은 **마크다운으로 만든 요소에 클래스·id·속성을 덧붙이는** 기능이다.

원래 마크다운으로 문단을 쓰면 이렇게 변환된다.

```text
안녕하세요
    ↓
<p>안녕하세요</p>
```

여기에 CSS 클래스를 붙이고 싶을 때 `{: ... }`를 쓴다.

```text
안녕하세요
{: .text-center }
    ↓
<p class="text-center">안녕하세요</p>
```

`{: }` 안에 넣는 것:

- `.클래스` → `class="클래스"` (점은 CSS 클래스 기호)
- `#아이디` → `id="아이디"` (샵은 id 기호)
- `key="값"` → `key="값"` (그 밖의 HTML 속성)

예를 들어 아래 문단은 `{: .text-center }`(Bootstrap 유틸)를 붙여 실제로 가운데 정렬된다. IAL은 이렇게 클래스 출처와 무관하게 어떤 클래스든 붙일 수 있다.

이 문단은 IAL로 가운데 정렬을 적용한 결과다.
{: .text-center }

참고로 앞으로 나올 콜아웃 `{: .prompt-tip }`, 이미지 옵션 `{: .shadow }` 도 전부 IAL이다.

### Chirpy 콜아웃 (prompt)

인용구 바로 다음 줄에 `{: .prompt-* }`를 붙인다.

> 팁 박스
{: .prompt-tip }

> 정보 박스
{: .prompt-info }

> 경고 박스
{: .prompt-warning }

> 위험 박스
{: .prompt-danger }

### 각주 (kramdown)

본문에 `[^식별자]`를 달고, 아무 곳에 `[^식별자]: 내용`으로 정의하면 페이지 하단에 각주가 모인다.

kramdown은 markdown 확장 엔진이다.[^kramdown] 각주 번호를 누르면 하단으로 이동한다.

[^kramdown]: Jekyll의 기본 마크다운 처리 엔진. 이 각주는 페이지 맨 아래에 표시된다.

### 코드 블록에 파일명 표시 (Chirpy)

코드 블록 다음 줄에 `{: file="..." }`를 붙이면 상단에 파일 경로가 표시된다.

```python
print("hello")
```
{: file="src/main.py" }

### 파일 경로 강조 (Chirpy)

인라인 코드에 `.filepath` 클래스를 주면 파일 경로 스타일이 된다.

설정은 `_config.yml`{: .filepath } 에서 바꾼다.


### 정의 목록 (kramdown)

용어와 정의를 `:` 로 연결한다.

kramdown
: 마크다운을 HTML로 변환하는 Jekyll 기본 엔진.

Chirpy
: 이 블로그가 쓰는 Jekyll 테마.
