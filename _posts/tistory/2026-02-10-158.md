### Usage Skills

#### 문서 스킬

pdf - pdf 파일을 구조화된 데이터로 읽고 쓴다.

doc - doc 파일을 구조화된 데이터로 읽고 쓴다.

spreadsheet - 엑셀 파일을 구조화된 데이터로 읽고 쓴다.

```
// 예시

$pdf sn74hc595.pdf 를 분석해서 $doc docs파일에 한글로 요약정리해줘

$spreadsheet 유형검사.xlsx 읽고 점수 계산하는 로직과 점수에 따른 유형 $doc 에 정리해줘
```

#### Tip

codex가 좋아져서 진행 현황을 중간중간에 브리핑해 준다. 보고 있으면

뭐 설치하려다 막혀서 뭘로 대체한다거나,

뭐 대체하다가 안 돼서 뭘 한다거나 하는 codex의 선택을 알려준다.

대안으로 진행해도 목적을 달성하긴 하는데

- 개발자의 의도를 따르자
- 대안의 대안, 또 그 대안으로 이어지며 방향이 과도하게 틀어지는 경우를 피하자

이러한 취지에서, 가능하다면 초기 계획대로 진행되도록 실패 요인을 해결하는 편이다.

실패 요인 해결 예시

- 필요한 환경 사용할 수 있도록 세팅 pip, node, apt-get, ...
- 네트워크나 접근 관련 권한 문제라면 "권한 올려서 진행해 줘"
- root가 필요하다면 "root 비밀번호 xxxx야 sudo apt-get ~~ 진행해 줘"

반복적인 실패 요인이 있다면, AGENTS.md 에 명시해 두는 것도 좋다.

작업 중간에 끼어들어 말할 때는 "Ctrl + c"로 중단하고 말하면 된다.

이전에는 중단하고 이어가면 기존 요청의 맥락이 일부 사라지는 느낌이 있었는데, 요즘에는 그런 현상이 많이 줄었다.

#### 노션 스킬

notion-knowledge-capture - 대화나 문서에서 핵심 인사이트를 추출해 Notion에 구조화하여 작성한다.

notion-research-documentation - 조사 결과를 정리해 Notion에 구조화하여 작성한다.

notion-meeting-notes - 내용을 논의/결정/액션 아이템 기준으로 정리해 Notion 회의록 템플릿으로 구조화하여 작성한다.

```
// 예시

$notion-knowledge-capture 지금 대화를 정리해서 노션에 작성해 줘
```

notion처럼 외부 시스템에 접근하는 경우, 필요한 MCP 연결을 자동으로 설정하고 인증 링크까지 준다.

MCP 연결 상태를 반영하기 위해 재시작이 필요하다는 안내가 있으면,

세션만 나갔다가 `codex resume --last`로 다시 들어와 진행하면 된다.
